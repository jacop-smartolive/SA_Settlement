# src/settlement/  [planned]

## 목적
**정산하기** — 기업 선택 → 정산 기간 선택 → 정산 실행. 실행 결과는 `../index.html` 업체 정산 완료 내역 목록에 레코드로 추가된다.

> 현재 미구현. 폴더만 선점해 둔 상태 (맥락 유지용).

## 예상 화면 구성
1. **기업 선택** — 검색/드롭다운 (한화에어로·두산·한화오션·HD현대 등)
2. **정산 기간 선택** — 시작월/종료월 (예: 2026-03)
3. **과금유형 자동 로드** — 기업의 기본 과금유형 프리셋 (정액제/1인당과금/이용료없음/구내식당)
4. **미리보기** — 예상 정산 금액/건수
5. **[정산 실행]** — 완료 레코드 생성 → 목록 페이지로 리다이렉트

## 예상 파일 (신설 시)
| 파일 | 역할 |
|---|---|
| `index.html` | 정산 실행 메인 화면 |
| `../assets/js/settlement-runner.js` | 집계·검증 로직 (신설) |
