개선 포인트 범례

1. 금액 우측 정렬 숫자 데이터 비교 가독성 향상. 자릿수가 맞아 한눈에 크기 비교 가능
2. 정산기간 연도 표시 연도(회색) + 월일(볼드)로 계층적 표시. 2019~2026 데이터 구분 가능
3. 현재 기간 하이라이트 최신 정산 행에 연한 빨강 배경 + 정산기간 빨강 텍스트로 즉시 식별
4. 버튼형 조회 토글 라디오 → 세그먼트 버튼형. 현재 모드가 직관적으로 보임
5. 삭제 버튼 위험 표시 hover 시 빨간색으로 위험 동작임을 시각적으로 경고
6. Zebra stripe 짝수 행 미세 배경색으로 행 추적 용이
7. 스크롤 힌트 오른쪽 끝 그라데이션으로 추가 열 존재를 시각적 암시
8. 요청일 연도 분리 연도를 작은 회색 텍스트로 분리하여 월.일 가독성 유지
