## 6. URL 매핑 표

| URL | 페이지 | 진입 방법 |
|---|---|---|
| `/` 또는 `/src/` | (디폴트 — index.html 리다이렉트 X) | 직접 입력 |
| `/src/index.html` | 정산 완료 내역 목록 | 진입점 |
| `/src/pages/settlement-detail.html?company=&count=` | 정산 기간별 조회 | 목록에서 클릭 |
| `/api/confirm-settlement?c=&r=&t=` | 정산서 확정 페이지 (HTML 응답) | 메일 본문 링크 |
| `/api/*` | API 엔드포인트 (JSON 응답) | fetch |

## 7. 페이지 이동 시 데이터 전달

### 목록 → 상세

```js
var qs = 'pages/settlement-detail.html?company=' + encodeURIComponent(r.name) + '&count=' + r.count;
```

- 회사명: URL 인코딩
- 정산 건수: 정수
- localStorage: 페이지 진입 시 자동 로드 (`v11-detail-settle-{기업명}` 등)

### 상세 → 메일

```js
MailSendModal.open({
  companyName: companyName,
  rowNo: rowNo,
  attachmentName: fileName,
  fetchAttachment: function() { ... },
  onSuccess: function(info) { recordMailSent(rowNo, info); }
});
```
