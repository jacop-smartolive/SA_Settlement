### 시트 1: '가맹점별 요약'
| 행 | 내용 | 핵심 셀 |
|---|---|---|
| R3 | 헤더 (사용기간, 구분, 가맹점명, ...) | - |
| R4~R24 | 21개 가맹점 명세 | B/C/D/E/F/G |
| R25 | 총계 | **F25=4375, G25=32595860** |
| R26 | 모바일 쿠폰 이용료 합계 | **G26=3223000** |
| R27 | 식대 이용료 합계 | **G27=29372860** |
| R28~R29 | 이용료 (사용인원×0원) | F29=168, G29=0 |
| R30 | 세금계산서 발행 금액 | G30=29372860 |
| R31 | 최종 정산 금액 | **G31=32595860** |
