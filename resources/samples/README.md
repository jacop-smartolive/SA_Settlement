# resources/samples/

## 목적
과금유형별 정산서 **엑셀 샘플/더미**. 코드가 참조하지 않으며 기획·테스트·비교용.

## 파일 목록
| 파일 | 과금유형 | 구분 |
|---|---|---|
| `settlement_flat_rate.xlsx` | 정액제 | 정산서 샘플 |
| `settlement_no_fee.xlsx` | 이용료없음 | 정산서 샘플 |
| `settlement_per_person.xlsx` | 1인당과금 | 정산서 샘플 |
| `dummy_정액제.xlsx` | 정액제 | 더미 데이터 |
| `dummy_1인당과금.xlsx` | 1인당과금 | 더미 데이터 |
| `기업명_20260000_정산서_정액제.xlsx` | 정액제 | 파일명 규칙 예시 |
| `기업명_20260000_정산서_이용료없음.xlsx` | 이용료없음 | 파일명 규칙 예시 |
| `기업명_202600_정산서_1인당과금.xlsx` | 1인당과금 | 파일명 규칙 예시 |
