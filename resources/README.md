# resources/ — 정적 리소스

> 코드가 fetch하는 외부 파일 모음

## 📁 구조

```
resources/
├─ samples/       # 정산서 Excel 샘플 (코드가 fetch)
│  └─ 디와이피엔에프(주)_2026-04-30_정산서.xlsx  ⭐ 공통 원본
├─ templates/     # 일괄등록 템플릿 (보조)
└─ references/    # 스크린샷·스냅샷 (참고용)
```

## ⭐ 핵심 파일

**`samples/디와이피엔에프(주)_2026-04-30_정산서.xlsx`**

모든 회사의 정산상세 페이지가 이 파일을 fetch하여 사용:
- 페이지 로드 시 SheetJS로 셀 추출 (G25, G26, G27, G31 등)
- 원본 다운로드 시 그대로 전송 (파일명만 회사별로 변경)
- 메일 발송 시 첨부
